R1700¶

Summary¶

The ‘Instantiated signal’ field for a Signal-type Event must be defined, whereas the ‘Called operation’ and ‘Expression’ fields must be empty.

Details¶

An Event is the specification of a specific occurrence at a specific point in space and time.

A Signal-type Event is meant to define a Signal, and cannot trigger an Operation at the same time.
