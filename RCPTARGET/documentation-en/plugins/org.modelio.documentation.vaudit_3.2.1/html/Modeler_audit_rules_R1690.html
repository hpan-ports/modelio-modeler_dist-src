R1690¶

Summary¶

The ‘Expression’ field for a Change-type Event must be defined, whereas the ‘Called operation’ and ‘Instanciated signal’ fields must be empty.

Details¶

Change-type Events are Events that define an expression.

Tips¶

Verify that the expression property is defined, and that ‘Instanciated signal’ and ‘Call operation’ fields are empty, or change the type of the Event.
