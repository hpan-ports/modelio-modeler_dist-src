R2650¶

Summary¶

The context of a protocol state machine must be a Classifier.

Details¶

A protocol state machine is always defined in the context of a Classifier. It specifies which operations of the Classifier can be called in which state and under which conditions, thus specifying the authorized call sequences on the Classifierâ€™s operations.

Tips¶

You should change the context of the state machine and choose a Classifier.
