Editing elements using edition boxes¶

Elements can be edited through an edition box by double-cliking on them, or by using the ‘ Edit element…’ contextual menu command.

Generic edition box¶

Key:

  1. UML tab, similar to the Element view.
  2. Audit tab, similar to the Check model window.
  3. Notes and constraints tab, similar to the Notes and constraints view.
  4. Modules tabs for each relevant module.
  5. Quick “Description” note edition frame.
