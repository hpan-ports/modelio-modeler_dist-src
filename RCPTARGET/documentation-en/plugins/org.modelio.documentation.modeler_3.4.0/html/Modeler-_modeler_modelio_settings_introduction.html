Introduction to Modelio settings¶

What are Modelio settings?¶

Modelio settings are quite simply parameters that you can define in order to adapt Modelio to your own personal preferences.

With Modelio, you can define the following settings that will apply to all your Modelio projects:

  • The default type used for attribute, parameter and return parameter creation
  • The log level
  • The LibreOffice/OpenOffice executables directory
  • The XMI Import/Export options

As well as these core settings, each Modelio module deployed in a project comes with a set of customizable parameters. For more information on individual module settings, please see the associated user guide.

Where do I define my Modelio settings?¶

Modelio settings are defined in the Settings window, accessed via the Configuration/Settings command.
