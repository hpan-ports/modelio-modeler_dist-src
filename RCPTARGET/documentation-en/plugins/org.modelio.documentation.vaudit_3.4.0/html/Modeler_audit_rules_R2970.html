R2970¶

Summary¶

An << assigned >> Dependency must be from an Actor, an Interface, a Package or a Process towards a Goal.

Details¶

An << assigned >> Dependency expresses the assigning of a Goal to an owner. This owner is typically an Actor, an Interface, Package (organizational unit) or Process.

Tips¶

If you have assigned a Goal to an element other than an Actor, a Package or a Process, you should delete your << assigned >> Dependency.
