generate-modelio-sources¶

Full name¶

org.modelio:modelio-maven-plugin:3.0.0.00:generate-modelio-sources

Description¶

Generates Java sources from a Modelio project. It executes the “Generate sources” command from Java Designer.

Binds by default to the generate-sources phase.

Required parameters¶

NameTypeSinceDescription
modelioDirFile2.2.0.01Modelio install to use.
Example: 
<modelioDir>C:\Program Files\modelio 3</modelioDir>
workspaceDirFile2.2.0.01Workspace to use.
Example: 
<workspaceDir>C:\Users\XXX\modelio\workspace</workspaceDir>
projectNameName2.2.0.01Project to use.
Example: 
<projectName>JavaProject</projectName>

Optional parameters¶

NameTypeSinceDescription
updateBeforeGenerationBoolean2.2.0.01If true, launches an SVN update before generating sources. The default value is 'false'.
Example: 
<updateBeforeGeneration>true</updateBeforeGeneration>
