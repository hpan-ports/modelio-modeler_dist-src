processCollaboration
Collaboration is within Modelio the root of BPMN process modeling. To the Modelio end user, Collaborations and processes mean the same thing. If there is only one pool in the collaboration, both are related (1-1). If there are several pools within the collaboration, then there will be one process per pool.
