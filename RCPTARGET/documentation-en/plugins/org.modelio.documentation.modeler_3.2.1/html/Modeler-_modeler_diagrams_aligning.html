Aligning elements¶

Modelio diagrams provide several different alignment possibilities. To align elements in a diagram, just select the elements you want to align and click on the desired icon in the diagram toolbar:

  • Align top: This aligns selected elements with the top of the element selected first.

  • Align left: This aligns selected elements with the left-hand side of the element selected first.

  • Align bottom: This aligns selected elements with the bottom of the element selected first.

  • Align right: This aligns selected elements with the right-hand side of the element selected first.

  • Align center: This horizontally aligns selected elements with the center point of the element selected first.

  • Align middle: This vertically aligns selected elements with the center point of the element selected first.

  • Set to the same height: This sets selected elements to the same height as the element selected first.

  • Set to the same width: This sets selected elements to the same width as the element selected first.

  • Set to the same size: This sets selected elements to the same size as the element selected first.
