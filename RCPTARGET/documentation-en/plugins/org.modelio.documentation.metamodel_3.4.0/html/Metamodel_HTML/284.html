AbstractPseudoState
Abstraction for PseudoState.
PseudoStates represent every kind of graphical Node in a StateMachine, except States, such as a branch, a fork, and so on. Modelio specific: AbstractPseudoState is used to seemlessly migrate old PseudoState elements to the upcoming UML2PseudoState class hierarchy.
