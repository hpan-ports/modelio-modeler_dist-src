AnalystProject
Topmost element of an analyst model tree in Modelio.

An analystProject in Modelio corresponds to the Analyst Model concept. An  AnalystProject instance in Modelio has several top-level elements which are roots holding several dedicated Analyst models (Goals, Requirement and so on). An AnalystProject instance does not belong to any other element.

Association
Description
PropertyRoot : PropertyContainer [1..1]
GoalRoot : GoalContainer [0..*]
BusinessRuleRoot : BusinessRuleContainer [0..*]
DictionaryRoot : Dictionary [0..*]
RequirementRoot : RequirementContainer [0..*]
GenericRoot : GenericAnalystContainer [0..*]
RiskRoot : RiskContainer [0..*]
