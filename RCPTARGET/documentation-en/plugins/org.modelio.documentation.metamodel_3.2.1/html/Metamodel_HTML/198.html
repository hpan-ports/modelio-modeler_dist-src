Node
Computational resource on which Artifacts may be deployed for execution. Nodes can be interconnected through communication paths to define network structures.
In the metamodel, a Node is a subclass of Classifier. Nodes may have an internal structure defined in terms of the Parts and Connectors associated with them, which is used to model the deployment of Artifact on Nodes, and also the imbrications of Nodes. In Modelio, Nodes belong to their owner NameSpace, which can be a Package, a Class or a Node.
