AttributeLink
Named slot in an instance, which holds the value of an Attribute.
An AttributeLink defines an Attribute's value at instance level. AttributeLinks appear attached to Instances or Parts. In UML 2.0, attribute links are called slots. In Modelio, an AttributeLink belongs to an Instance.
