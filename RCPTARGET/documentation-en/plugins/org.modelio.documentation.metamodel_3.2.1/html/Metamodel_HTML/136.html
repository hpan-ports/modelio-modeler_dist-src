MatrixDefinition
Definition of a matrix. A matrix is defined by a lines columns and the value for a line and a column. Line and column content are each defined by a query definition. The matrix values are defined by a MatrixValueDefniition.
Figure 119 : MatrixDefinition
Association
Description

Lines definition.
ColumnsDefinition : QueryDefinition [0..1]
Columns definition. If not specified, the line definition is used.

Matrix values definition.

Depth definition. May be left undefined.
