Dictionary
Dictionary
A Dictionary can contain Terms and other Dictionaries. Its purpose can be described in the Text attribute. A Dictionary belongs to its parent Dictionary, except for the root Dictionary, which belongs to the AnalystProject. A Dictionary can be typed. In this case, all owned dictionaries and terms must have the same type. It then also has a default PropertyValueSet, which stores all PropertySet property default values (see PropertyValue). This default PropertyValueSet is then used to initialize new Term property values and Dictionary default values.
