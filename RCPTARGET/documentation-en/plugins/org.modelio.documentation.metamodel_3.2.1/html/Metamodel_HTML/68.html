BpmnIntermediateCatchEvent
the Intermediate Event indicates where something happens (an Event) somewhere between the start and end of a Process. It will affect the flow of the Process, but will not start or (directly) terminate the Process. Intermediate Events can be used to: * Show where Messages are expected or sent within the Process, * Show delays are expected within the Process, * Disrupt the normal flow through exception handling, or * Show the extra work required for compensation.
