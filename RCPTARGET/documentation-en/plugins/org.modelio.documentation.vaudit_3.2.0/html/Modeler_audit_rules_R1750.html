R1750¶

Summary¶

Repetition of names is forbidden for all AtrributeLinks.

Details¶

An AttributeLink is the value of an Attribute and must have the same name as the Attribute it instanciates. This name must be unique in the instanciated Classifier as well as in the I_nstance_.

Tips¶

Verify that all the slots have the same name as the attributes they instanciate, and if they do, make sure all attributes have a unique name in the instanciated Classifier.
