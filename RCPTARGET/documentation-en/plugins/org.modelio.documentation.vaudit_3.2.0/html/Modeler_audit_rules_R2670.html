R2670¶

Summary¶

A protocol state machine cannot have history vertices.

Details¶

A protocol state machine specifies which operations of the classifier can be called in which state and under which conditions, thus specifying the authorized call sequences on the classifierâ€™s operations. Therefore history vertices should not be used in a protocol state machine.

Tips¶

Delete any history vertices from the protocol state machine, or move them to

another state machine (not protocol).
